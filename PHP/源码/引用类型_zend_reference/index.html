еј•з”Ёзұ»еһӢ_zend_reference

еҸ‘иЎЁдәҺ 2020-12-09 жӣҙж–°дәҺ 2021-03-18 еҲҶзұ»дәҺ PHP пјҢ жәҗз Ғ 
<?php
$a = "string";
$b = &$a;
echo $a;
echo $b;

$b = "hello!";
echo $a;
echo $b;

unset($b);
echo $b;
echo $a;

4гҖҒ5иЎҢиҫ“еҮә$aе’Ң$bйғҪжҳҜеј•з”Ёзұ»еһӢпјҢ$bеј•з”Ё$aпјҢдҪҶжҳҜ$aд№ҹдјҡеҸҳжҲҗеј•з”Ёзұ»еһӢ zend_referenceпјҢ$aе’Ң$bзҡ„zvalйғҪжҢҮеҗ‘еҗҢдёҖдёӘzend_referenceгҖӮzend_referenceдёӯзҡ„zvalдёӯзҡ„u1.type=6,иЎЁзӨәеӯ—з¬ҰдёІзұ»еһӢ, p *z.value.ref.val.value.str.val@6 string

8гҖҒ9иЎҢ$aе’Ң$bжҢҮеҗ‘зҡ„zend_referenceдёӯзҡ„zvalзұ»еһӢиҝҳжҳҜеӯ—з¬ҰдёІ,дҪҶжҳҜжү“еҚ°еҮәжқҘе·Із»ҸйғҪжҳҜhelle!дәҶ

12иЎҢ $bжҢҮеҗ‘зҡ„zvalжҳҫзӨәзұ»еһӢжҳҜ0,д№ҹе°ұжҳҜIS_UNDEF

13иЎҢ $a зҡ„zvalзҡ„typeдҫқз„¶жҳҜ10(еј•з”Ёзұ»еһӢ), жү“еҚ°д»Қз„¶жҳҜhello!

еҸҜд»ҘзңӢеҮә, 11иЎҢunset($b)зҡ„ж“ҚдҪң,д»…д»…жҳҜжҠҠ$bзҡ„zvalдёӯзҡ„typeз”ұ10ж”№жҲҗ0, е…¶д»–зҡ„е®Ңе…ЁдёҚеҠЁгҖӮ
