PHPеһғеңҫеӣһж”¶жңәеҲ¶

еҸ‘иЎЁдәҺ 2021-02-02 жӣҙж–°дәҺ 2021-03-24 еҲҶзұ»дәҺ PHP пјҢ жәҗз Ғ

еј•з”Ёи®Ўж•°

  1. жҜҸдёӘеҶ…еӯҳеҜ№иұЎйғҪеҲҶй…ҚдёҖдёӘи®Ўж•°еҷЁпјҢеҪ“еҶ…еӯҳеҜ№иұЎиў«еҸҳйҮҸеј•з”Ёж—¶пјҢи®Ўж•°еҷЁ+1
  2. еҪ“еҸҳйҮҸеј•з”Ёж’ӨжҺүеҗҺпјҢи®Ўж•°еҷЁ-1
  3. е®ҡжңҹжЈҖжҹҘеҗ„еҶ…еӯҳеҜ№иұЎзҡ„и®Ўж•°еҷЁпјҢеҰӮжһңи®Ўж•°еҷЁ=0пјҢиЎЁжҳҺеҶ…еӯҳеҜ№иұЎжІЎжңүиў«дҪҝз”ЁпјҢиҜҘеҶ…еӯҳеҜ№иұЎеҲҷиҝӣиЎҢй”ҖжҜҒ
  4. еһғеңҫеӣһж”¶е®ҢжҲҗ

еӣӣиүІж Үи®°

зӨәдҫӢпјҡ

<?php
$a = [];		# и®Ўж•°еҷЁ = 1
$a[] = &$a;		# и®Ўж•°еҷЁ = 2
	
unset($a);		# и®Ўж•°еҷЁ = 1

еҜ№иұЎе’Ңж•°з»„пјҢжңүеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳпјҢй’ҲеҜ№иҝҷдёӨз§Қж•°жҚ®зұ»еһӢпјҢйҮҮз”ЁеӣӣиүІж Үи®°жі•иҝӣиЎҢеһғеңҫеӣһж”¶

й»‘иүІпјҡжӯЈеёёж•°жҚ®
зҙ«иүІпјҡз–‘дјјеһғеңҫ
зҒ°иүІпјҡ
зҷҪиүІпјҡеһғеңҫ

  1. е°Ҷи®Ўж•°еҷЁ -1 иҝҮе№¶дё”зӣ®еүҚдёҚдёә 0 зҡ„ж•°з»„ж”ҫеҲ°зј“еҶІеҢәпјҢе№¶е°Ҷе…¶ж Үи®°дёәзҙ«иүІ
  2. еҜ№зј“еҶІеҢәдёӯзҡ„ж•°з»„иҝӣиЎҢж·ұеәҰйҒҚеҺҶпјҢе°Ҷзҙ«иүІзҡ„е…ғзҙ ж Үи®°дёәзҒ°иүІпјҢе№¶и®Ўж•°еҷЁ -1
    жіЁж„ҸпјҡеҸӘеҜ№е…ғзҙ иҝӣиЎҢж ҮзҒ°гҖҒ-1ж“ҚдҪң
  3. еҶҚж¬Ўж·ұеәҰжү«жҸҸпјҢжЈҖжҹҘзҒ°иүІзҡ„е…ғзҙ пјҢеҰӮжһңи®Ўж•°еҷЁдёҚдёә0пјҢе°Ҷе…¶ж Үи®°дёәй»‘иүІе№¶и®Ўж•°еҷЁ+1пјҲдёҠдёҖжӯҘжңү-1ж“ҚдҪңпјүпјҢеҰӮжһңи®Ўж•°еҷЁдёә0пјҢе°Ҷе…¶ж Үи®°дёәзҷҪиүІ
  4. жү«жҸҸж•°з»„пјҢй»‘иүІзҡ„е…ғзҙ д»Һзј“еҶІеҢәдёӯз§»йҷӨпјҢзҷҪиүІзҡ„е…ғзҙ и®Ўж•°еҷЁ+1е№¶з§»еҲ° to_free еҲ—иЎЁ
  5. жё…йҷӨ to_freeпјҢе®ҢжҲҗеһғеңҫеӣһж”¶

з®ҖеҚ•иҜҙпјҡе°Ҷз–‘дјјеһғеңҫзҡ„ж•°з»„зҡ„е…ғзҙ зҡ„и®Ўж•°еҷЁ-1пјҢ=0иҜҙжҳҺдёәеһғеңҫпјҢвү 0иҜҙжҳҺдёҚжҳҜеһғеңҫ
