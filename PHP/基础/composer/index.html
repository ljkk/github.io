composer

еҸ‘иЎЁдәҺ 2020-12-09 жӣҙж–°дәҺ 2021-03-24 еҲҶзұ»дәҺ PHP пјҢ еҹәзЎҖ

дёӯж–ҮзҪ‘з«ҷпјҡhttps://pkg.phpcomposer.com/

composer.jsonдёӯи®°еҪ•ејҖеҸ‘иҖ…дё»еҠЁе®үиЈ…зҡ„еҢ…, composer.lockдёӯйҷӨдәҶи®°еҪ•composer.jsonдёӯи®°еҪ•зҡ„еҢ…д№ӢеӨ–, иҝҳи®°еҪ•composer.jsonдёӯи®°еҪ•зҡ„еҢ…зҡ„дҫқиө–е’Ңдҫқиө–зҡ„дҫқиө–зӯүзӯү; жүҖд»ҘиҜҙcomposer.lockдёӯи®°еҪ•зҡ„жҳҜжңҖе…Ёзҡ„

windowsдёӢе®үиЈ…composer

  1. жүҫеҲ°е№¶иҝӣе…Ҙ PHP зҡ„е®үиЈ…зӣ®еҪ•

  2. е°Ҷ composer.phar еӨҚеҲ¶еҲ° PHP зҡ„е®үиЈ…зӣ®еҪ•дёӢйқўпјҢд№ҹе°ұжҳҜе’Ң php.exe еңЁеҗҢдёҖзә§зӣ®еҪ•гҖӮ

    • (composer.pharзҡ„дёӢиҪҪй“ҫжҺҘпјҡhttps://getcomposer.org/download/)

  4. еңЁ PHP е®үиЈ…зӣ®еҪ•дёӢж–°е»әдёҖдёӘ composer.bat ж–Үд»¶пјҢе№¶е°ҶдёӢеҲ—д»Јз ҒдҝқеӯҳеҲ°жӯӨж–Үд»¶дёӯгҖӮ

    • @php вҖң%~dp0composer.pharвҖқ %*

е‘Ҫд»Ө

еҰӮжһңжң¬ең°еӯҳеңЁcomposer.lockж–Үд»¶, еҲҷжЈҖжҹҘжӣҙж–°composer.lockдёӯзҡ„еҢ…, еҰӮжһңжң¬ең°дёҚеӯҳеңЁcomposer.json, еҲҷжЈҖжҹҘжӣҙж–°composer.json

жЈҖжҹҘжӣҙж–°composer.json, жіЁж„ҸиҝҷдёӘжҳҜжЈҖжҹҘжӣҙж–°жүҖжңүеҢ…, еҰӮжһңеҸӘжғіжӣҙж–°жҢҮе®ҡзҡ„еҢ…, еҸҜд»Ҙе‘Ҫд»ӨеҗҺйқўжҢҮе®ҡеҢ…, дҫӢеҰӮ:

composer update topthink/framework

дёҚжҳҜеҫҲжҺЁиҚҗиҝҷдёӘж–№ејҸ, жҺЁиҚҗдҪҝз”Ёrequire

е®үиЈ…иЈ…жҲ–иҖ…жӣҙж–°еҢ…

жӣҙж–°autoloadи®ҫзҪ®, жү§иЎҢиҝҷжқЎе‘Ҫд»ӨеҗҺ, еҸҜд»ҘжҹҘзңӢ vender/composer/autoload_psr4.phpжҹҘзңӢжӣҙж”№пјҢз„¶еҗҺеҸӘиҰҒдҝқиҜҒеј•е…Ҙ vender/autoload.php еҚіеҸҜгҖӮ

еҜ№дәҺжҹҗдәӣеҢ…пјҢдҪңиҖ…еҸҜиғҪжІЎжңүеҸ‘еёғиҜҒд№ҰзүҲжң¬пјҢеҰӮжһңиҝҷдёӘж—¶еҖҷдҪ йңҖиҰҒдҪҝз”ЁиҝҷдёӘеҢ…пјҢдҫӢеҰӮ

composer require easyswoole/oss
