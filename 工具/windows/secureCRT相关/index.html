1. 超时断线

secureCRT 无操作过一段时间会自动断线, 很麻烦

解决办法:

Option -> Global Option -> Edit Default Setting … -> Terminal -> 勾选 Send protocol NO-OP every 60s

2. 菜单消失

解决办法: alt + enter
