cat

еҰӮжһңжІЎжңүжҢҮе®ҡж–Үд»¶пјҢжҲ–иҖ…жҢҮе®ҡж–Үд»¶дёә вҖң-вҖқпјҢеҲҷд»Һж ҮеҮҶиҫ“е…ҘиҜ»еҸ–

-n    # еҜ№иҫ“еҮәеҶ…е®№ж ҮиЎҢеҸ·
-b    # еҜ№иҫ“еҮәеҶ…е®№йқһз©әиЎҢж ҮиЎҢеҸ·
-E    # жҳҫзӨәиЎҢз»“жқҹз¬Ұ$

-s    # еҺӢзј©иҝһз»ӯзҡ„з©әиЎҢжҲҗдёҖиЎҢ

tr

translateпјҢиҪ¬жҚўгҖҒеҲ йҷӨгҖҒеҺ»йҮҚж ҮеҮҶиҫ“е…Ҙдёӯзҡ„еӯ—з¬ҰпјҢе°Ҷз»“жһңеҶҷе…ҘеҲ°ж ҮеҮҶиҫ“еҮә

tr [OPTION]... SET1 [SET2]

-d    #еҲ йҷӨ
-c    #еҸ–иЎҘйӣҶ
-s    #еҜ№set1иҝӣиЎҢеҺ»йҮҚж“ҚдҪң
-t

trзұ»дјјдәҺsedе‘Ҫд»ӨпјҢдҪҶжҳҜжҜ”sedз®ҖеҚ•пјҢжүҖжңү tr иғҪе®һзҺ°зҡ„еҠҹиғҪпјҢsed йғҪиғҪе®һзҺ°

зӨәдҫӢпјҡ

# е°ҶaжҚўжҲҗb
$tr a b
# з”ҹжҲҗ8дҪҚйҡҸжңәж•°
$ cat /dev/urandom | tr -dc A-Za-z0-9 | head -c8
1imvyPoH
# е°Ҷa.logдёӯеӨ§еҶҷиҫ“еҮәдёәе°ҸеҶҷ
$ cat a.log
HELLO WORD
$ tr A-z a-z <a.log
hello word
