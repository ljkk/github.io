дёәд»Җд№Ҳ32дҪҚcpuеҸӘж”ҜжҢҒ4GеҶ…еӯҳ?

еҮҶзЎ®зҡ„иҜҙжҳҜcpuзҡ„ең°еқҖжҖ»зәҝдҪҚе®Ҫдёә32дҪҚзҡ„ж—¶еҖҷжңҖеӨ§еҸӘиғҪж”ҜжҢҒ4Gзҡ„еҜ»еқҖ

еҜ»еқҖе°ұжҳҜCPUжңҖеӨ§иғҪжҹҘжүҫеӨҡеӨ§иҢғеӣҙзҡ„ең°еқҖгҖӮCPUзҡ„еҜ»еқҖд»Ҙеӯ—иҠӮдёәеҚ•дҪҚ пјҲеӯ—иҠӮжҳҜжңҖе°ҸеҸҜеҜ»еқҖеҚ•дҪҚпјү

32дҪҚCPUдёҖж¬ЎеҸӘиғҪеӨ„зҗҶ32дҪҚзҡ„ж•°жҚ®пјҲе°ұжҳҜ4дёӘеӯ—иҠӮпјүпјҢ2зҡ„32ж¬Ўж–№bit=4GпјҢжүҖд»Ҙ32дҪҚCPUж”ҜжҢҒзҡ„жңҖеӨ§еҜ»еқҖз©әй—ҙе°ұжҳҜ4GпјҲдёҚиҖғиҷ‘дҪҝз”ЁPAEвҖ”вҖ”зү©зҗҶең°еқҖжү©еұ•жҠҖжңҜпјүпјҢеҰӮжһңжҸ’дәҶдёҖж №8Gзҡ„еҶ…еӯҳжқЎпјҢи¶…иҝҮ4Gзҡ„еҶ…еӯҳйғЁеҲҶи·ҹдёҚз”ЁдёҚеҲ°

CиҜӯиЁҖдёӯеҗ„дёӘеҸҳйҮҸеҚ з”Ёзҡ„еҶ…еӯҳеӨ§е°Ҹ

int еҚ з”Ёеӣӣеӯ—иҠӮпјҢ32дҪҚпјҢиҢғеӣҙ -2^31 \~ 2^31-1пјҢиҖҢ long int ж №жҚ®зј–иҜ‘еҷЁдёҚеҗҢпјҢиЎЁзӨәзҡ„иҢғеӣҙд»Һ32дҪҚеҲ°64дҪҚпјҢе®ғеҸҜд»ҘдҝқиҜҒиҮіе°‘32дҪҚгҖӮ

long int дёҚжҳҜж ҮеҮҶзҡ„64дҪҚж•ҙеһӢпјҢиҖҢlong long жҳҜпјҢе®ғеңЁжүҖжңүзҡ„зј–иҜ‘еҷЁдёӯйғҪиЎЁзӨә64дҪҚж•ҙеһӢпјҢиЎЁзӨәиҢғеӣҙ -2^63 ~ 2^63-1

е…ідәҺ __int64пјҡ
еңЁC/C++дёӯпјҢ64дёәж•ҙеһӢдёҖзӣҙжҳҜдёҖз§ҚжІЎжңүзЎ®е®ҡи§„иҢғзҡ„ж•°жҚ®зұ»еһӢгҖӮзҺ°д»Ҡдё»жөҒзҡ„зј–иҜ‘еҷЁдёӯпјҢеҜ№64дёәж•ҙеһӢзҡ„ж”ҜжҢҒд№ҹжҳҜж ҮеҮҶдёҚдёҖпјҢеҪўжҖҒеҗ„ејӮгҖӮдёҖиҲ¬жқҘиҜҙпјҢ64дҪҚж•ҙеһӢзҡ„е®ҡд№үж–№ејҸжңүlong longе’Ң__int64дёӨз§Қ(VCиҝҳж”ҜжҢҒ_int64)пјҢиҖҢиҫ“еҮәеҲ°ж ҮеҮҶиҫ“еҮәж–№ејҸжңүprintf(вҖң%lldвҖқ,a)пјҢprintf(вҖң%I64dвҖқ,a)пјҢе’Ңcout << aдёүз§Қж–№ејҸгҖӮ

е…ідәҺfloatе’Ңdoubleзҡ„иЎЁзӨәиҢғеӣҙе’ҢзІҫеәҰпјҡ

float: 1bit(з¬ҰеҸ·) + 8bit(жҢҮж•°дҪҚ) + 23bit(е°ҫж•°дҪҚ)
double: 1bit(з¬ҰеҸ·) + 11bit(жҢҮж•°дҪҚ) + 52bit(е°ҫж•°дҪҚ)

жө®зӮ№ж•°еҖј = (з¬ҰеҸ·)(1.е°ҫж•°) * (2)^(еёҰз¬ҰеҸ·зҡ„жҢҮж•°)
дёҠиҝ°е…¬ејҸзҡ„1е’Ң2е°ұжҳҜиҝҷд№Ҳи§„е®ҡзҡ„пјҢдёҚеҝ…еӨӘеңЁж„Ҹ
